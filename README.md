## BOILERPLATE REACT-ES6-WEBPACK-WITH-LIVE-EDIT

## Installation
npm install

## Build
webpack

## Run
npm start

**goto loacalhost:3000**

## Features
* React es2015 with react-router

* Webpack tooling with live editing (hot-reload)

* Very simple http-proxy set up for eventual use with API
